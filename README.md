# Decoding the "Critical Eye": A Multilingual Analysis of Hospitality Brands

## 📌 Project Overview
This project investigates how linguistic and cultural backgrounds influence customer satisfaction in the luxury hotel industry. By analyzing 80,000+ reviews in English and French, I identified a significant "Brand Expectation Gap" between Canadian and French hotel chains.

## 🚀 Key Insights
* **The Critical Eye:** French-language reviews are statistically more critical of service quality than English reviews, suggesting a need for cultural weighting in global KPIs.
* **Brand Expectation Gap:** French travelers hold national brands (e.g., Sofitel) to a higher service standard than foreign "prestige" brands (e.g., Fairmont).
* **Linguistic Priorities:** Discovered that French-speaking guests prioritize culinary details and room comfort, whereas English-speaking guests prioritize location and staff efficiency.

## 🛠️ Technical Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, Seaborn, NLTK/Regex (for custom language detection)
* **Techniques:** NLP, Sentiment Analysis, Comparative Brand Analytics
